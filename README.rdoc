== Rspec for Starters

== Getting Started

1. From the terminal, clone the repository:
      git clone git@github.com:mikeebert/rspec_for_starters_too.git

2. Switch to application folder:
      cd rspec_for_starters_too

3. Bundle
      bundle install

4. Test if rspec works:
      rspec spec
